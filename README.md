# Middle Korean AI (NMT-MK)
중세국어 신경망 기계 번역 시스템 개발 프로젝트

## 프로젝트 개요
본 프로젝트는 15세기에서 17세기 사이의 중세국어(옛한글) 문헌을 현대어로 정교하게 번역하는 AI 모델을 구축하는 것을 목표로 합니다. 데이터가 부족한 중세국어의 특성을 극복하기 위해 대규모 언어 모델을 통한 데이터 증강과 역사 문헌 데이터베이스 크롤링 파이프라인을 구축합니다.

## 기술 로드맵 및 사양
하드웨어 인프라
- CPU: AMD Ryzen 9 9900X
- GPU: NVIDIA RTX 5080 (16GB VRAM)
- RAM: 64GB DDR5

모델 아키텍처
- 베이스 모델: Gemma 4 26B MoE (4-bit QLoRA)
- 학습 프레임워크: Unsloth (효율적인 메모리 사용 및 학습 속도 최적화)
- 텍스트 표준화: NFD (자모 분리) 기반 유니코드 정규화

## 디렉토리 구조
- configs/: 학습 하이퍼파라미터 및 설정 파일
- data/: 초도 수집 데이터 및 정규화 완료된 코퍼스
- src/:
  - augment/: LLM 기반 합성 데이터 생성 로직
  - crawlers/: 국사편찬위원회 등 역사 DB 크롤러
  - models/: 학습 및 추론 엔진
  - preprocess/: 옛한글 정규화 및 토크나이저 확장
  - utils/: 공통 유틸리티
- notebooks/: 실험용 주피터 노트북

## 현재 진행 상황
- 프로젝트 구조화 및 초기 아키텍처 수립 완료
- 국사편찬위원회(조선왕조실록 등) 크롤러 스켈레톤 구축
- 합성 데이터 생성을 위한 프롬프트 엔지니어링 수행 중
- 유니코드 NFD 표준화 및 방점 처리 로직 설계

## 향후 계획
1. 병렬 코퍼스 수집 파이프라인 고도화 및 데이터 클렌징
2. Gemma 4 기반 중세국어 이해도 사전 평가
3. Unsloth를 활용한 QLoRA 파인튜닝 실험 시작
4. 옛한글 전용 토크나이저 확장 및 임베딩 최적화
