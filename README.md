# Middle Korean AI (NMT-MK)
중세국어 신경망 기계 번역 시스템 개발 프로젝트

## 프로젝트 개요
본 프로젝트는 15세기에서 17세기 사이의 중세국어(옛한글) 문헌을 현대어로 정교하게 번역하는 AI 모델을 구축하는 것을 목표로 합니다. 데이터가 부족한 중세국어의 특성을 극복하기 위해 대규모 언어 모델을 통한 데이터 증강과 역사 문헌 데이터베이스 크롤링 파이프라인을 구축합니다.

## 데이터 소스 (Data Sources)
본 프로젝트는 다음의 신뢰도 높은 역사 문헌 데이터베이스를 주요 소스로 활용합니다.
- 국사편찬위원회 (NIKH): 조선왕조실록, 승정원일기 등 국역 및 원문 데이터
- 한국학중앙연구원 장서각: 왕실 도서관 소장 디지털 문헌
- Open Korean Historical Corpus (OKHC): Hugging Face(seyoungsong)에 공개된 중세국어 병렬 말뭉치 스냅샷

## 기술 로드맵 및 사양
하드웨어 인프라
- CPU: AMD Ryzen 9 9900X
- GPU: NVIDIA RTX 5080 (16GB VRAM)
- RAM: 64GB DDR5

모델 아키텍처
- 베이스 모델: Gemma-2-9B-IT (4-bit QLoRA)
- 학습 프레임워크: PyTorch + HF PEFT (Windows 환경 안정성 확보)
- 텍스트 표준화: NFD (자모 분리) 기반 유니코드 정규화

## 디렉토리 구조
- configs/: 학습 하이퍼파라미터 및 설정 파일
- data/: 초도 수집 데이터 및 정규화 완료된 코퍼스
- src/:
  - api/: FastAPI 웹 서비스 및 프론트엔드 정적 파일
  - augment/: LLM 기반 합성 데이터 생성 로직
  - crawlers/: 국사편찬위원회 등 역사 DB 크롤러
  - models/: 학습 및 추론 엔진 (Inference v3)
  - preprocess/: 옛한글 정규화 및 데이터셋 빌더
  - utils/: 공통 유틸리티
- lora_model_bidirectional_v3/: 최종 양방향 번역 LoRA 어댑터

## 현재 진행 상황
- **Phase 7 완료**: 양방향(중세-현대) 번역 모델 학습 및 웹 대시보드(localhost:8000) 배포 완료.
- **데이터 증강**: 864쌍의 고품질 양방향 지시어 데이터셋 구축 및 학습 성공.
- **UI 구현**: 프리미엄 다크 테마 기반의 반응형 웹 인터페이스 탑재.

## 향후 계획 (Phase 8)
1. **역사적 환상(Hallucination) 제어**: 구한말 행정 문서 노이즈 필터링 및 15세기 언해본 데이터 집중 재학습.
2. **지시어 정교화**: 15세기 세종/세조 시대 문체(옛한글) 고정을 위한 시스템 프롬프트 가드레일 강화.
3. **용어 의역 엔진**: 현대적 개념을 중세 어휘로 자연스럽게 변환하는 의역 로직 고도화.
